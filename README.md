# Bhow Bhow Pet Clinic Website

A responsive static website based on the earlier Homeo clinic project and adapted for Bhow Bhow Pet Clinic, Vijayawada.

## Run locally

Open `index.html` directly, or start a local server in this folder:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.
